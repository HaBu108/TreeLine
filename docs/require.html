System Requirements

Linux

    The stable version of TreeLine (2.0.x) requires the following libraries and programs:
  • Python (Version 3.2 or higher)
  • Qt (Version 4.6, 4.7 or 4.8)
  • PyQt (Version 4.8.x, 4.9.x, 4.10.x or 4.11.x)
  • If spell checking capability is desired, either the aspell, ispell or hunspell programs are required

Windows

Should run on any computer running Windows XP, Vista, 7 or 8.

If spell checking capability is desired, an external program is required. Either aspell, ispell or hunspell must be installed.
